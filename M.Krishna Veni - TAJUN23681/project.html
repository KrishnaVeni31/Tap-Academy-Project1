Mini Project Work

Online Leave Management System

In This Project, Leave Management System is to develop a web based application using which management of leave application and approval is done online. Faculty member of the institute can apply for leave and adjust their load directly from this application. Manager of their respective department can grant the leave application.

Major Project Work

Machine Learning Algorithm for Brain Stroke Detection

A machine learning algorithm for brain stroke detection is a computer program that can analyze medical data and identify signs of a stroke in the brain. By training on large datasets of brain images and associated stroke cases, the algorithm learns to recognize patterns and make accurate predictions. Its purpose is to assist healthcare professionals in diagnosing strokes quickly and accurately, enabling prompt medical intervention for better patient outcomes.
